# Archive Directory

This directory contains old scripts and examples that are kept for reference purposes.

## Contents

- **client_example.py**: Example of using Graphiti client directly
- **test_simple_search.py**: Simple search functionality test
- **test_search_citations.py**: Citation search test
- **analyze_relations.py**: Utility for analyzing graph relationships

## Why these are archived

These files are not part of the main codebase anymore but are kept for:
- Reference implementation patterns
- Historical context
- Debugging and troubleshooting

## Note

If you need to use any of these scripts, they may require updates to work with the current codebase.
